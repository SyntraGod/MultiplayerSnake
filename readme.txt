Mô tả phần mềm<Game Snake>:
-Khi bật game sẽ có lựa chọn đăng nhập. 
	+ Nếu đã có tài khoản chọn sign in
	+ chưa có chọn sign up để tạo tài khoản -> sau khi tạo xong sẽ quay trở lại cửa sổ đăng nhập, đăng nhập để vào menu
	+ Exit dể tắt cửa sổ game
- Sau khi đăng nhập cửa sổ menu hiện lên:
	+ Sign out: Đăng xuất khỏi tài khoản hiện tại
 	+ Sigle player: Hiện lên cửa sổ game, người chơi sẽ điều khiển con rắn bằng cách sử dụng 4 nút w,a,s,d tương ứng cho rắn
	di chuyển lên trên,trái,dưới,phải. Người chơi điều khiển rắn ăn các cục thức ăn xuất hiện ngẫu nhiên trên bản đồ để tăng 
	điểm của mình. Màn chơi sẽ kết thúc khi người chơi điều khiển rắn va vào bản thân. Lúc này cửa sổ sẽ hiện lên điểm số của
	bản thân.
	+ Multi player: Chơi multiplayer, tối đa 2 người chơi
	+ HighScore: Cho phép người chơi xem bảng xếp hạng điểm của server
	+ Exit: tắt cửa sổ game
